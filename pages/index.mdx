# 关于

使用**Next.js** 和 [Nextra](https://nextra.vercel.app/)库构建，可以自行美化和扩展web界面，部署和备份相当简单。

花几分钟[**部署自己的**](https://vercel.com/new/git/external?repository-url=https://github.com/vercel-solutions/nextjs-portfolio-starter&project-name=portfolio&repository-name=portfolio)网页。

在此项目的GITHUB库中创建文档，随后他会展示在帖子中。

Vercel 在自动building时如果md文件中存在`<<` 或者 `>>`会building失败，正如该页面所显示需要使用符号 `` 才会building成功。

如果被构建的文件中图片使用的是Html样式，如：`<img src="https://xxx/img/xxx.png" alt="name" style="zoom:80%;" />` ，vercel会构建失败，必须是md样式引入图片，如`![name](https://xxx/img/xxx.png)`。
